📋 الخطوة 3: النشر على Netlify (10 دقائق)
───────────────────────────────────────────────────────────────

الطريقة A: عبر GitHub (الأفضل)

1. رفع الكود على GitHub:
   cd /workspace/promohive
   git init
   git add .
   git commit -m "PromoHive v1.0 - Ready for deployment"
   
   # Create repo on github.com first, then:
   git remote add origin https://github.com/YOUR_USERNAME/promohive.git
   git push -u origin main

2. في Netlify:
   - افتح: https://app.netlify.com/
   - اضغط "Add new site"
   - اختر "Import an existing project"
   - اختر "Deploy with GitHub"
   - اختر repo: promohive
   - Build command: npm run build
   - Publish directory: build
   - اضغط "Deploy site"

3. إضافة Environment Variables:
   - Site settings → Environment variables
   - أضف:
     * VITE_SUPABASE_URL = https://jtxmijnxrgcwjvtdlgxy.supabase.co
     * VITE_SUPABASE_ANON_KEY = [من Supabase API settings]
   - Trigger deploy

───────────────────────────────────────────────────────────────
الطريقة B: Drag & Drop (للاختبار السريع)

1. Build محلياً:
   cd /workspace/promohive
   npm install
   npm run build

2. رفع:
   - افتح: https://app.netlify.com/drop
   - اسحب مجلد "build" إلى الصفحة

⚠️ لاحقاً: أضف Environment Variables كما في الطريقة A
